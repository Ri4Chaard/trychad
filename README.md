# Chad

- User Registration and Login: User data is managed globally using Zustand to simulate registration and login functionalities.
- Form Validation: Zod and React Hook Form are employed to handle form validation for consistent, error-free input management.
- Animations: Framer Motion is integrated for smooth, fluid animations.
- Responsive Design: Tailwind CSS is used for consistent styling, providing responsive layouts and utility classes.

# Login page
- The "Join the waitlist" button on the login page redirects users to the registration page.
